Welcome to Igniter.js Javascript Butter!

Igniter.js gives you the power to get thing done in a very small package... The minified version is around 5k! Just include the js file and you're ready to start coding... Igniter.js uses jquery like syntax and selectors.



hide, show, toggle, click, listen

   
$i(selector).hide();                
$i(selector).show();
$i(selector).toggle();
$i(selector).click(callback);
$i(selector).listen(event,callback);
If you can see this text the script his working... Click on the buttons to hide, show and toggle this text.

To save space, I wrote one method listen to handle all javascript events, instead of writing a method for each event. I did write the click method just because it is used so much, I thought it was worth the extra space...



Code Example


<span class="toggle-text"> this text.</span>

<button class="btn btn-secondary" id="hideBtn">Toggle</button>
<button class="btn btn-success" id="showBtn">Show</button>
<button class="btn btn-primary" id="toggleBtn">Toggle</button>


$i("#hideBtn").listen("click",function(){
        $i(".toggle-text").hide();
});

$i("#showBtn").listen("click",function(){
        $i(".toggle-text").show();
});

$i("#toggleBtn").click(function(){
        $i(".toggle-text").toggle();
});
